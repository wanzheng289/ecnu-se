# # WEB3目前主要有哪几个大的推手（大厂、社区、人物等均可）希望能够推动web3的话语权越大越好？
## 一、背景
上世纪90年代到千禧年的静态互联网WEB1是互联网的初级形态，比如雅虎和早期的搜狐等。在这一阶段，网络内容由少数专业创作者生产，用户只能被动浏览，无法参与内容创作或互动。随着技术的发展，到了2000年左右，互联网进入WEB2阶段。用户不再仅仅是内容的消费者，更成为了内容的生产者，可以在博客平台写文章、在社交网站分享生活、在视频平台上传作品。而互联网的第三次重大演变WEB3则去除了对平台的中心化，用户不再依赖单一平台，而是通过加密技术和分布式节点，实现对自身数字身份、创作内容和虚拟资产的完全掌控。它基于区块链、智能合约等技术将数据所有权和治理权交还给个人，构建了一个用户真正拥有数据和资产的去中心化网络。此外，WEB3还强调计算机能够理解内容的含义和上下文，通过使用NLP和ML等技术提供更智能化的搜索结果和信息过滤，并提供更加定制化的信息服务和用户体验，比如脸书的图谱搜索就使用了自然语言处理来搜索和结合用户的社交图谱数据。
WEB3的话语权更多地体现在行业协议标准、资金流、生态建设和政策等方面。WEB3的大推手往往对核心代码库有较大贡献，并在定义行业底层协议和标准、模块化区块链架构、安全审计、去中心化定序器开发等方面具有较大的话语权。此外，资金的流向从某种程度上体现了整个行业的风向标。比如随着比特币、以太坊及山寨币ETF的普及，华尔街巨头的资金流入规模成为衡量话语权的关键。此外，开发者和真实活跃用户的数量也是衡量一个生态是否具有大的话语权的标准。截至2025年初，全球超5.6亿人拥有加密货币并使用WEB3工具；而召车应用与送餐服务商Grab和稳定币发行商StraitsX签署谅解备忘录，其旗下的GrabPay商户将能接受来自新加坡和国际消费者的稳定币支付，这些支付将通过广泛使用的Web3或数码钱包进行，旨在探索在亚洲范围内开发基于Web3的支付基础设施，这种与亿级用户入口的结合代表了巨大的生态话语权。此外，随着部分国家将比特币纳入战略储备，以及各国对稳定币的落地，政府与立法者在WEB3规则制定中的话语权显著提升。
## 二、科技大厂
在WEB3的影响力逐渐扩大的今天，许多科技巨头为WEB3的发展做出了极大贡献。比如，Microsoft通过Microsoft Entra Verified ID大力推动了去中心化身份的落地，并且其旗下的云计算服务平台Microsoft Azure深度整合了Quorum区块链服务，并与Ankr和zkSync等协议合作，极大降低了部署企业级节点的门槛。此外，Google Cloud通过Blockchain Node Engine为开发者提供稳定且合规的底层支持，已经成为以太坊、Solana和EigenLayer等多个主流公链的重要验证节点和基础设施合作伙伴。
许多大厂对WEB3的基础设施建设提供了极大的支持。比如，亚马逊云AWS利用其全球市场份额，通过Managed Blockchain服务承载了大量WEB3协议的后端，并在今年优化了零知识证明的计算加速，为Layer 2的扩容方案提供了较大的算力支持。此外，阿里云作为业内领先的云计算服务提供商，依托高速可靠的全球网络和强大的计算资源，不仅助力构建区块链基础设施，还支持高效、安全的WEB3应用开发，在去年与Conflux宣布携手合作，致力于为香港企业提供本土化的WEB3行业解决方案，重点围绕零售、旅游、娱乐、艺术、教育等行业提供支持。
## 三、企业和机构
在最新的产品发布会上，Base宣布了其生态系统的重大升级，将战略重心从纯粹的底层基础设施转向了一个集用户、应用、社交于一体的超级应用。这次转型的核心是将Coinbase Wallet升级并更名为Base App，将其品牌从单一的链升级为一个包括Base Chain、Base Build和Base App的开放技术堆栈。这一战略转型反映了Base及其背后的某交易平台对当前行业格局的深刻洞察，它们不再满足于仅做一个应用部署平台，而是希望成为用户进入WEB3世界的首选界面。
2023年，全球最大交易所币安在土耳其伊斯坦布尔举办币安区块链周上，宣布推出内建在币安App的自我托管加密货币钱包——币安WEB3钱包，旨在提供更简单方便的产品，让用户安全体验WEB3世界，不需要担心会弄丢助记词，或是处理超复杂的注册流程。所谓自我托管，就是用户不需要依靠第三方托管或中间机构，对于自己的资产、财产有掌控权，好比自己做自己的银行，不用担心个人隐私或第三方的安全性，但用户自己也要对资产安全负最大的责任。截至今年年底，币安依然维持着极高的市场份额，其旗下的BNB Chain在DeFi生态增长中处于领先地位，并成功向现实世界资产RWA和AI应用领域持续扩张。
企业不仅是资金的聚集地，而且在定义WEB3技术标准上具有较大话语权。作为MetaMask的母公司，ConsenSys掌控着数千万用户的WEB3入口，并通过以太坊L2方案Linea深度地参与了对公链话语权的竞争，在开发者工具、身份认证和去中心化标准制定上拥有较大的发言权。此外，a16z在今年发布的《加密行业现状》报告中阐述了“金融机构全面拥抱加密货”、“稳定币走向主流”、“区块链基础设施（几乎）已为黄金时段做好准备”和“加密货币与人工智能正在融合”等主要发现，反映了其投资逻辑从对技术的追逐向构建政策网络和关系网络的转变，致力于在全球范围内争取更有利的监管话语权。
## 四、社区与组织
开发者是WEB3的核心生产力。从某种程度上来说，哪个开发者社区能吸引更多开发者，哪个开发者社区就掌握了对底层协议的定义权。在Github上，WEB3的开源生态表现出极高的韧性。根据Electric Capital 2025的开发者报告，以太坊以超过三万余名活跃开发者稳居首位，几乎是Solana的两倍。此外，Gitcoin在今年完成了其在战略上的转型。通过Gitcoin Grants 23实验，Gitcoin首次在大规模范围内同时运行平方募资与追溯性资助，分发了超130万美元的资金。这种模式让开发者不再依赖大厂，而是直接由社区投票决定资金去向，极大地增强了社区的主导话语权。
作为一种基于区块链技术的组织形式，去中心化自治组织DAO通过智能合约实现自治、透明和去中心化的管理，旨在通过区块链技术和智能合约来取代传统的公司治理模式，其核心在于去中心化和自治，所有的决策和操作都由组织成员共同参与，而不是由中心化的管理层控制。今年MakerDAO的Endgame计划进入了关键阶段，推出了全新的品牌形象及USDS和SKY代币，并通过成立多个子DAO实现了治理的专业化分工，还引入了AI工具协助决策，旨在将其稳定币体系推广为全球金融的去中心化基石。此外，Uniswap DAO治理在今年通过了涉及销毁高达1亿枚代币的UNIfication提案，并通过DAO投票直接决定了协议收入的分配，展示了去中心化治理在商业决策中的实际话语权。
## 五、人物
在技术创新方面，以太坊联合创始人Vitalik Buterin深度主导了以太坊向The Verge和The Purge阶段的跨越，并大力推动零知识证明在隐私保护和身份验证中的应用，其影响力已经从单纯的编程延伸到加速去中心化以及AI与Crypto的融合。Polkadot的创始人Gavin Wood在2025年推出了Join-Accumulate Machine架构，试图打破区块链的不可能三角——在区块链设计中无法同时实现去中心化、安全性和可扩展性；其核心理念在于通过个性证明来建立无需中介的数字信任体系。
在商业方面，Coinbase的CEOBrian Armstrong在今年积极倡导将预测市场作为政策制定的信号工具，并领导Coinbase通过Base链实现了交易所向全链应用平台的转型。此外，币安的创始人赵长鹏以个人身份通过区块链教育和对AI及生物技术的投资，依然在BNB Chain生态和全球开发者社区中维持着巨大的影响力。
在理论研究方面，a16z Crypto的合伙人Chris Dixon通过其著作和资本定义了Read-Write-Own的WEB3标准，成功游说大量传统开发者转向区块链建设。此外，著名的WEB3意见领袖Punk6529是一位重要的思想领袖和加密艺术的倡导者，最初于2021年7月作为NFT收集者出现在加密货币推特上，现在已经通过他们的推文转变为一个以任务驱动的构建者。Memes NFT系列旨在推广6529的开放元宇宙原则，例如去中心化、社区、自我主权等。
