#### 可能为0吗？
$假设对于AES的S盒，输入差分为0x1，输出差分可能为0$
$\therefore S(X)\oplus S(X \oplus 0x01)=0$
$\therefore S(X)=S(X\oplus 0x01)$
$\therefore X和X\oplus 0x01的输出相同$
$又\because X\neq X\oplus 0x01$
$\therefore 两个不同的输入产生了相同的输出$
$又\because AES\quad S盒是双射函数$
$\therefore 不同的输入产生不同的输出$
$\therefore 矛盾$
$\therefore 假设不成立$

### 问题2：对于DES的$S_1$盒，若输入差分为0x1，则输出差分有几种可能取值？可能为0吗？取到每种输出差分的概率相同吗？

#### 若输入差分为0x1，则输出差分有几种可能取值？
由于DES的S盒为6bit输入，所以需要遍历$2^6=64$种X的取值，对每个X计算与之配对的输入$X'=X \oplus 0x01$,然后计算输出差分$\Delta X=S(X)\oplus S(X')$
```python
DES_S1_BOX=[
[14,  4, 13, 1,  2, 15, 11, 8, 3, 10, 6, 12,  5, 9, 0, 7],
[ 0, 15,  7, 4, 14,  2, 13, 1,10,  6,12, 11,  9, 5, 3, 8],
[ 4,  1, 14, 8, 13,  6,  2,11,15, 12, 9,  7,  3,10, 5, 0],
[15, 12,  8, 2,  4,  9,  1, 7, 5, 11, 3, 14, 10, 0, 6,13],
]
delta_in=0x01 #输入差分
output_differences=set()

def s1_lookup(input6bit):
    #提取每一位
    b5=(input6bit>>5)&0b1
    b4=(input6bit>>4)&0b1
    b3=(input6bit>>3)&0b1
    b2=(input6bit>>2)&0b1
    b1=(input6bit>>1)&0b1
    b0=input6bit&0b1
    row=(b5<<1)|b0 #行号-最高位和最低位
    col=(b4<<3)|(b3<<2)|(b2<<1)|b1 #中间4位是列号
    return DES_S1_BOX[row][col]

for x in range(64): #遍历
    x2=x^delta_in
    if x2>=64:
        continue
    y1=s1_lookup(x)
    y2=s1_lookup(x2)
    delta_out=y1^y2
    output_differences.add(delta_out)
    
print(f"输入差分共{len(output_differences)}种")
print(sorted(output_differences))
```
输出：
输入差分共11种
[3, 5, 6, 7, 9, 10, 11, 12, 13, 14, 15]

#### 可能为0吗？
$\because\ 输入为6\text{bit}，输出为4\text{bit}，即~\{0,1\}^6 \rightarrow \{0,1\}^4$
$\therefore \exists X \neq X',S(X)=S(X')$
$令X'=X\oplus \Delta X,S(X)=S(X')$
$\therefore \Delta Y=S(X)\oplus S(X')=0$
$\therefore \exists \Delta X \neq 0,\Delta Y =0$
$又\because 由上一段代码的输出结果可知，当输入差分=0x01时，输出差分列表中并没有0$
$\therefore 当输入差分=0x01时，输出差分不可能为0，但是存在一种输入差分，使得输出差分为0$
